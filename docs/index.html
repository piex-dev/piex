PieX — Extend Pi without forking

Core capabilities extracted from oh-my-pi, Claude Code, OpenCode and other coding agents, packaged as independent @piex-dev/* npm packages. Install with pi install.

Get Started GitHub
pi — piex 
$ pi install npm:@piex-dev/hashline
$ pi install npm:@piex-dev/dap
$ pi install npm:@piex-dev/lsp

Design Principles

100% Extension API based. Independent versioning. Traceable origins.

🔌

Extension API only

Register tools, commands & hooks. Never touch the Pi core.

📦

Install on demand

Only install hashline, dap, plan — whatever you need.

🔖

Independent releases

Pi upgrades never break piex. Each package has its own npm version.

🧬

Traceable origins

Every capability credits its source (omp, pi examples, community themes, etc).

📦
