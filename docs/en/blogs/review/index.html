Review: First-Class Code Review for Agents

Repeatable review, not paste-a-diff improvisation.

💡

In short

/review turns “paste a diff for AI” into a product: clear scope, filtered noise, one engine for humans and models.

Background

Ad-hoc “please review this” fails on scope, noise, prompt drift, and split human/model entry points.

@piex-dev/review unifies /review and a callable tool on one diff engine.

How it works

Modes for working tree, staged, base branch, commit, or custom instructions. Filter locks/build/vendor/media; skip huge raw diffs.

Implementation

Single-file lightweight port of omp review without multi-agent overlays.

Roadmap

Structured findings, LSP/DAP hooks, project ignores, multi-pass, later subagents.
