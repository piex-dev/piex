Dark Terminal Theme: Make pi Look Like a Real Terminal

Skin only. Logic unchanged.

💡

In short

Want a serious hacker-terminal look instead of generic blue-gray? Install Dark Terminal, switch in /settings. Behavior stays the same.

Background

You stare at the agent TUI all day. High-contrast terminal aesthetics should be a skin, not a behavior change.

@piex-dev/theme-dark-terminal is a pure JSON theme via pi.themes.

How it works

No TS entry. vars palette maps into required semantic color tokens: green accent, blue structure, cyan info, red error, yellow warning, near-black backgrounds.

Implementation

Ported from opencode-themes/dark-terminal. Prefer absolute paths for global installs so reload keeps the theme. Zero hooks.

Roadmap

Soft and color-blind variants, token docs, visual fixtures, bulk import pipeline.
