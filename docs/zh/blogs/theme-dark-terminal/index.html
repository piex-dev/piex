2026-07-19 Theme TUI UX

Dark Terminal 主题：把 pi 涂成一台真正的终端

只换皮，不改 agent 逻辑。

💡

导语

想把 pi 看成一台认真的黑客终端，而不是通用灰蓝面板：装上 Dark Terminal，/settings 切一下即可，行为逻辑零改变。

问题背景

Coding agent TUI 是长时间注视的界面。需要高对比暗终端美学时，应只换皮、不改行为。

@piex-dev/theme-dark-terminal 是纯 JSON 主题包，经 pi.themes 分发。

技术原理

无 TS 入口：package.json 挂 themes 目录。vars 原子色 → colors 语义 token。绿强调、蓝结构、青信息、红错误、黄警告、近黑背景。

实现方案

自 opencode-themes/dark-terminal 映射到 pi schema。全局安装注意绝对路径，避免 reload 丢主题。零 hook、零网络。

设计参考

项目机制piex 取舍
opencode-themes dark-terminal51 color token 高对比暗终端配色；bg: #050505、green/blue/red 强调色系采纳：色彩角色与语义映射。适配：pi 额外 token（thinking gradient 强度色 #ff0088 为 pi 侧增量）
pi theme schemavars → colors 两层映射；pi.themes JSON 分发；零代码热加载完全遵循：只提供静态 JSON，不写 TS、不走 hook

核心取舍：不做代码、不改行为、不绑宿主终端；只把 opencode-themes 的色板翻译到 pi 的 token 名上。

优化计划

soft / 色弱变体；token 对照表；视觉回归；批量从 opencode-themes 导入。
